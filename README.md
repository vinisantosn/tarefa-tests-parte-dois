# Testes Unitários e TDD - ClienteService

Este repositório contém um projeto Java desenvolvido com o intuito de demonstrar a prática de Test-Driven Development (TDD) e testes unitários utilizando JUnit.

## Objetivo

O objetivo deste projeto é exercitar a criação de testes unitários e a metodologia TDD (Test-Driven Development) em um cenário de desenvolvimento de software. O código fonte contém classes para manipulação de clientes (`ClienteService` e `ClienteDao`) e testes unitários para validar o comportamento dessas classes.

## Estrutura do Projeto

- **`src/main/java/edu/ebac/service`**: Este pacote contém a classe `ClienteService`, responsável por fornecer funcionalidades relacionadas à manipulação de clientes.
- **`src/main/java/edu/ebac/dao`**: Neste pacote, temos as interfaces e implementações das classes relacionadas ao acesso aos dados dos clientes.
- **`src/test/java`**: Este diretório contém os testes unitários para validar o comportamento da classe `ClienteService`.

## Como Executar os Testes

Para executar os testes unitários, você pode utilizar uma IDE Java compatível com JUnit, como Eclipse ou IntelliJ IDEA, ou executá-los via linha de comando utilizando o Maven. Certifique-se de que o JUnit esteja configurado corretamente no seu ambiente de desenvolvimento.
